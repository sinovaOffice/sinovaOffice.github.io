APLICACIÓN AUTOMATIZADA DE CUMPLEAÑOS
Descripción general:
Con esta aplicación se verifican las personas que cumplen años en cada día y se envía de forma automática una plantilla de mensaje predefinido con una imagen a todas ellas. En la interfaz inicial se listan las personas que cumplen años durante el mes actual.

CERTIFICADOS LABORALES
Descripción general:
Mediante esta aplicación se generan certificaciones laborales con opciones disponibles para el usuario y descarga del archivo en formato PDF para una mayor portabilidad.

APLICACIÓN DE SOLICITUD DE VACACIONES
Descripción general:
Mediante esta aplicación los colaboradores de una empresa pueden solicitar sus días hábiles de vacaciones, con configuración de un máximo de días que se pueden solicitar y con notificaciones al email tanto para el administrador como para el solicitante, una vez se agregan solicitudes, se aprueban, devuelven o rechazan las mismas.

APLICACIÓN DE SOLICITUD DE PERMISOS
Descripción general:
Mediante esta aplicación se solicitan permisos al jefe inmediato por un lapso de minutos u horas.
